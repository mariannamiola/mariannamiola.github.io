# Resources

A collection of software tools, datasets, and references related to my research in computational geometry, geostatistics, and environmental modeling.

---

## Software I Develop

<ul class="entry-list">

  <li>
    <div class="entry-title"><a href="https://github.com/mariannamiola/MUSEgeom">MUSE-geometry</a></div>
    Geospatially-aware C++ tool for 2D/3D modeling of complex geometries using mesh-based techniques. Standalone geometry processing module extracted from MUSE. Public repository.
  </li>

  <li>
    <div class="entry-title"><a href="https://github.com/DanielaCabiddu/EWOPE">EWoPe — Embeddable Workflow Persistence</a></div>
    Lightweight C++ library for tracking computational workflow metadata, designed to ensure reproducibility of multi-stage geoscientific pipelines. Public repository.
  </li>

  <li>
    <div class="entry-title">MUSE — Modeling Uncertainty as a Support for Environment</div>
    Stochastic software for spatial uncertainty modeling on unstructured 2D/3D meshes, integrating geostatistics and geometry processing. TRL = 9. Available by request.
  </li>

</ul>

## Useful Libraries & Tools

<ul class="entry-list">

  <li>
    <div class="entry-title"><a href="https://github.com/mlivesu/cinolib">CinoLib</a></div>
    C++ header-only library for polygonal and polyhedral mesh processing. SGP 2024 Software Award winner. Lead developer: Marco Livesu (CNR-IMATI).
  </li>

  <li>
    <div class="entry-title"><a href="https://www.paraview.org">ParaView</a></div>
    Open-source scientific visualization tool. Extensively used for 3D rendering of unstructured meshes and stochastic simulation results via both GUI and Python scripting.
  </li>

  <li>
    <div class="entry-title"><a href="https://www.meshlab.net">MeshLab</a></div>
    Open-source mesh processing tool for editing, cleaning, and visualizing 3D triangular and polygonal meshes.
  </li>

  <li>
    <div class="entry-title"><a href="https://qgis.org">QGIS</a></div>
    Open-source geographic information system for managing, visualizing, and analyzing geospatial vector and raster data.
  </li>

  <li>
    <div class="entry-title"><a href="https://www.cs.cmu.edu/~quake/triangle.html">Triangle</a></div>
    C library for generating high-quality triangular meshes via constrained Delaunay triangulation. Widely used in computational geometry.
  </li>

  <li>
    <div class="entry-title"><a href="https://tetgen.berlios.de">TetGen</a></div>
    Software for generating quality tetrahedral meshes from boundary representations. Used for volumetric geological model generation.
  </li>

</ul>
